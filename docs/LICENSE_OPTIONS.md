# License options before publication

This repository candidate intentionally does not choose an open-source license for the author.

Recommended options:

## MIT

Simple permissive license. Good for adoption and GitHub visibility.

## Apache-2.0

Permissive, includes explicit patent grant. Often preferred for infrastructure/devtool projects.

## AGPL/GPL

Copyleft. Stronger sharing requirements, but may reduce enterprise adoption.

## MIT-licensed/custom

Possible if the goal is public review but restricted commercial reuse.

## Current candidate status

Until a final license is chosen, treat the code as MIT License.
